Esta documentación ha sido generada automáticamente usando **MkDocs** y material visual *Material Design*.

El objetivo documentar la arquitectura **Page Object Model (POM)** construida sobre Selenium y Python para probar la interfaz del Instituto de Astrofísica de Canarias.

### Estructura de Navegación
En el panel izquierdo puedes encontrar el desglose técnico de cada una de las "Páginas" u objetos de automatización creados, con un nivel de detalle en sus parámetros, tipos y utilidades para que cualquier programador pueda reutilizarlos en otros tests sin investigar el código fuente.
